# WordStatistics

A simple Python program that reads a text file and provides basic statistics about the words in it.  
---

## Features

- Count total words in a file
- Count unique words
- Calculate average word length
- Show top 10 word frequencies
- Show 5 longest words
- Show 5 shortest words

---

## How to Run

1. Open your terminal or command prompt.
2. Navigate to the folder where the program is saved.
3. Run the program using:

```bash
python word_statistics.py sample_file.txt
```

### Requirements
No dependencies for external packages.
